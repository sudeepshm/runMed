# PharmaGuard Monorepo

This repository contains the source code for the PharmaGuard application.

## Structure

- `backend01/`: Python FastAPI backend.
- `frontend01/`: Next.js frontend.

## Getting Started

1. **Backend**:
   ```bash
   cd backend01
   # See backend01/README.md
   ```

2. **Frontend**:
   ```bash
   cd frontend01
   # See frontend01/README.md
   ```
